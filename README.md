# 🎮 Modern Rock Paper Scissors

A modern and interactive Rock Paper Scissors game built using **Python** and **Tkinter**.  
This project features a clean dark gradient UI, custom rounded buttons, hover animations, and a Best-of-5 match system.

---

## 🚀 Features

- 🎨 Modern gradient user interface
- 🔘 Custom rounded animated buttons
- ✨ Hover effects
- 🏆 Best-of-5 match logic
- 📊 Live score tracking
- 🔄 Reset functionality
- 🧠 Clean and structured Python code

---

## 🛠️ Technologies Used

- Python
- Tkinter (GUI Library)
- OOP Concepts
- Event Handling
